# Options & Greeks

<LanguageSelector />

ThetaDataDx provides comprehensive options data access, including full chain retrieval, Greeks computation, implied volatility, open interest analysis, and volatility surface construction.

## Capabilities

| Feature | Description |
|---------|-------------|
| **Option Chains** | Full chain by expiration, filtered by strike/type |
| **Greeks** | Delta, gamma, theta, vega, rho |
| **Implied Volatility** | Per-strike and per-expiration IV |
| **Open Interest** | Current and historical OI by strike |
| **Volatility Surface** | 3D vol surface construction from market data |
