# Getting Started

<LanguageSelector />

ThetaDataDx is a direct-wire SDK for accessing [ThetaData](https://thetadata.us/) market data without the overhead of their desktop terminal. It provides native clients for **Rust**, **Python**, **Go**, and **C++**.

## Why ThetaDataDx?

- **No Terminal Required** -- Connect directly to ThetaData's REST and streaming APIs
- **Multi-Language** -- Idiomatic SDKs for Rust, Python, Go, and C++
- **Zero-Copy Parsing** -- Responses are parsed with minimal allocations (Rust/C++)
- **Streaming Built-In** -- First-class WebSocket support with automatic reconnection
- **CLI & Tools** -- Query data from the command line, serve via REST, or use with AI via MCP

## Prerequisites

- A [ThetaData](https://thetadata.us/) subscription (Standard or Pro)
- Your ThetaData API credentials
