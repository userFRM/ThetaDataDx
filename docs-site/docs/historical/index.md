# Historical Data

<LanguageSelector />

ThetaDataDx provides access to ThetaData's comprehensive historical market data, including end-of-day summaries, intraday OHLC bars, and tick-level trades and quotes.

## Data Types

| Type | Description | Granularity |
|------|-------------|-------------|
| **End-of-Day** | Daily OHLC, volume, open interest | 1 day |
| **Intraday Bars** | OHLC bars at configurable intervals | 1 min -- 1 hour |
| **Tick-Level** | Individual trades and NBBO quotes | Tick |
| **Bulk Snapshots** | Full-market snapshots at a point in time | Configurable |

## Sections

- [End-of-Day](/historical/eod) -- Daily bars and summaries
- [Intraday Bars](/historical/intraday) -- Minute, 5-minute, hourly bars
- [Tick-Level](/historical/tick-level) -- Trade and quote ticks
